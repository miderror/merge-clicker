# Merge Clicker Game

## О проекте
Игра в жанре "кликер" с элементами объединения предметов (Merge) и прогрессией. Игрок сбрасывает кирки для разрушения блоков, зарабатывает игровую валюту и улучшает инструменты.

### Ключевые механики:
* Система инвентаря с возможностью слияния (объединения) предметов.
* Система уровней и генерации блоков.
* Сохранение игрового прогресса между сессиями.
* Поддержка управления на PC и мобильных устройствах.

---

## Архитектура проекта

### 1. Сохранение данных (Save System)
Реализована расширяемая система сохранения через единый интерфейс:
```csharp
public interface ISaveSystem
{
    void SaveData();
    void LoadData();
}
```
* **SaveManager** автоматически регистрирует и оповещает подписанные системы.
* Каждая игровая система самостоятельно отвечает за сериализацию собственных данных.

### 2. Игровые менеджеры
Используется паттерн Singleton для обеспечения глобального доступа к ключевым компонентам (например, `GameManager`) без создания жестких перекрестных связей в коде.

### 3. Генерация блоков
Применяется детерминированная генерация уровней на основе сида (seed), зависящего от текущего уровня игрока, что обеспечивает идентичную последовательность блоков.

## Сборка и запуск
1. Откройте проект в Unity (версия 2022.3 или новее).
2. Перейдите в настройки сборки (Build Settings) и выберите целевую платформу (PC / Android / WebGL).
3. Запустите сцену `SampleScene` из папки `Assets/Scenes/`.
